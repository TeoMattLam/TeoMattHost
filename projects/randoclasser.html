RandoClasser is a random seating plan generator for school classes. It is made in Kotlin and it is temporarily only available for Windows. It features:

  • Custom desk placement
  • Custom student list
  • Grouping students that must or must not sit together
  • Placing students in a specific row
  • Error logging

How To Use

Start by opening the tab labeled "Rows", where you can add rows by clicking the leftmost button in the bottom toolbar.
You start counting rows from the front of the class (marked by the teacher's desk) to the back. You can add a new desk with a custom number of seats.
To change the direction the class is facing, you have to change the teacher's desk position from the toolbar.
Next, go to the "Students" tab, where you can add students by clicking the second button in the toolbar. By doing that you create a student card. Each card has five rectangles on it. The first one is the student's name, which can't be changed. The second one is the row where the student must seat. Then you have the student's deskmate. Finally, you can select the rows and the students the student mustn't sit in or with. You can leave everything blank if you don't need any of these conditions.
The only thing left to do is press "Start". If you want to change something you can simply press on someone's name and then someone else's to swap them.
You might receive red error messages if you put too many conditions that can't be satisfied. For example, if you have 5 students and you put that 3 of them must sit in the first row, which only has 2 seats.
